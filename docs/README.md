# library-web — docs

Documentation for the web SPA (React + TanStack). Talks to the backend over REST; types are
generated from `library-backend/contract/openapi.yaml` via `openapi-typescript`. Follows a
strict TanStack-only / no-React-hooks discipline.

```
docs/
├── sow/   Statement of Work
├── prd/   Product Requirements
├── fsd/   Functional Spec
├── plan/  Implementation Plan
└── adr/   Architecture Decision Records
```
